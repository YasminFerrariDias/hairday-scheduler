# HairDay Scheduler

Sistema de agendamento de horários desenvolvido em JavaScript, permitindo selecionar data, horário e cliente, com exibição dinâmica dos agendamentos.

---

## 🚀 Funcionalidades

* Seleção de data para agendamento
* Escolha de horários disponíveis (manhã, tarde e noite)
* Cadastro de cliente
* Criação de novos agendamentos
* Listagem de horários organizados por período
* Cancelamento de agendamentos
* Atualização dinâmica da interface

---

## 🧠 Conceitos aplicados

* Manipulação do DOM
* Organização modular de código
* Separação de responsabilidades (modules, services, utils)
* Eventos (click, submit, load)
* Consumo de serviços (API/local)
* Estruturação de projeto com Webpack

---

## 📂 Estrutura do projeto

* `src/modules` → funcionalidades da aplicação
* `src/services` → comunicação com API
* `src/utils` → funções auxiliares
* `src/libs` → bibliotecas externas
* `src/styles` → estilos
* `index.html` → estrutura da aplicação
